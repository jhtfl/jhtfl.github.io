Instruction to play SMTV in tiles on Windows platform

  • Download the latest releases of ffmpeg and yt-dlp and put them on your PATH.
  • Create a new batch file named something like smtv.bat, and inside it, place following contents:

    • @echo off

    set ww=%1
    set hh=%2

    set url="https://www.youtube.com/watch?v=ZzWBpGwKoaI"
    set useragent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36"
    set format=93
    set upres=3

    @echo on

    :loop

    yt-dlp "%url%" --user-agent "%useragent%" -4 -f "%format%" -o - | ffplay - -vf "scale=w=iw*%upres%/%ww%:h=ih*%upres%/%hh%,tile=%ww%x%hh%" -autoexit -loglevel quiet -fs

    goto loop

  • Run smtv.bat with column number and row number. For example: "smtv.bat 4 3" will play SMTV in tiles as 4 columns and 3 rows

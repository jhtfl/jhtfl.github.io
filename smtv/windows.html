Instruction to play SMTV in tiles on Windows platform

  • Please refer to SMTV Livestream Tiling for indepth details.
  • Download the latest releases of ffmpeg and yt-dlp and put them on your PATH.
  • Create a batch file named something like smtv.bat with following contents:

    • @echo off

    set ww=%1
    set hh=%2

    set "url=https://www.youtube.com/watch?v=ZzWBpGwKoaI"
    set "useragent=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/115.0.0.0 Safari/537.36"
    set format=91
    set upres=7.5
    rem Use the following For clear picture
    rem set format=93
    rem set upres=3

    yt-dlp "%url%" --user-agent "%useragent%" -4 -f "%format%" -o - | ffplay - -vf "scale=w=iw*%upres%/%ww%:h=ih*%upres%/%hh%,fps=source_fps*%ww%*%hh%,tile=%ww%x%hh%" -autoexit -loglevel quiet

  • Create another batch file named something like smtv-run.bat with following contents:

    • @echo off

    smtv 4 3

  • Run smtv-run.bat which will play SMTV in tiles as 4 columns and 3 rows. You may modify numbers in the batch file to suite your needs
